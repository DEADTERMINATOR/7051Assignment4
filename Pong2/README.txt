Control for player 1:
 up : d-pad up , W
 down : d-pad down, S

Control for player 2:
 up : d-pad up, up arrow
 down : d-pad down, down arrow

Restart game when ended : start or space

Bring up console : C

Console commands : 
  QUIT - exit console
  RED - Make background red
  BLUE - Make console blue
  SPEEDUP - speed up ball and paddles
  SPEEDDOWN - speed down ball and paddles
